# CLI Tetris
An implementation of Tetris that can be played in the terminal.

## Dependencies
This project has the following system dependencies:
- Python 3

## Setup
- `./setup.sh`

## Configuration
To configure the game, modify `settings.py`

## Run game
- `./main.py`

## Run tests
- `nosetests`

## Run linter
- `flake8`
